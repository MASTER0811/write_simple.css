# CSS Api

## Getting Started

Insert this link tag in head element

```html
  <link rel="stylesheet" href="https://raw.githubusercontent.com/MASTER0811/css-api/main/style.css">
```


## Only Main Page
1. Main height
  `height: 100vh;`
```html
  Exp:<body class="bd_h"></body>
```
2. Main width
  `width: 100%;`
```html
  Exp:<body class="bd_w"></body>
```
3. Make all center
  `display: flex;`
  ` align-items: center;`
  `justify-content: center;` 
```html
  Exp:<body class="allct"></body>
```
